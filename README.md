## Ramp Up
Ramp Upは、学習記録を可視化するアプリです。手軽に学習時間を記録して、振り返りをすることができます。

## 技術スタック

#### 言語・フレームワーク
- Nust.js @nuxt/cli v2.14.7

## 機能一覧
- 新規登録・ログイン・ログアウト機能
- スキル(学習項目)設定機能
- 学習時間投稿機能
- 翌日のタスクを書き込む機能
- カレンダー表示機能
- グラフ表示機能
- アカウント編集機能
- スキル編集機能

## 環境構築
#### VueCLIのインストール

```
$ npm install -g @vue/cli
```

## 起動
```
$ yarn run dev
http://localhost:8000
```
